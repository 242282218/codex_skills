# 交互设计模式

## 按钮交互

### 状态
- 默认: 正常显示
- 悬停: 背景变深 + 阴影增加
- 活跃: 按下效果 + 缩放 0.98
- 禁用: 透明度 0.5 + 光标 not-allowed
- 加载: 显示加载动画 + 禁用交互

### 动画
```css
transition: all 150ms cubic-bezier(0.4, 0, 0.2, 1);
```

## 表单交互

### 输入框
- 焦点: 边框色变为主色 + 阴影
- 错误: 边框色变为危险色 + 错误提示
- 成功: 边框色变为成功色 + 成功图标
- 禁用: 背景变灰 + 光标 not-allowed

### 验证反馈
- 实时验证: 输入时检查
- 错误提示: 红色文字 + 图标
- 成功提示: 绿色文字 + 对勾
- 加载状态: 显示加载中提示

## 列表交互

### 行悬停
- 背景色变化
- 操作按钮显示
- 阴影增加
- 过渡时间: 150ms

### 选择
- 复选框状态变化
- 行高亮显示
- 批量操作按钮出现
- 计数器更新

## 模态框交互

### 打开
- 背景淡入 (150ms)
- 内容从下往上滑入 (300ms)
- 焦点转移到模态框

### 关闭
- 内容从上往下滑出 (300ms)
- 背景淡出 (150ms)
- 焦点恢复到触发元素

## 通知反馈

### 类型
- 成功: 绿色 + 对勾图标
- 错误: 红色 + 叉号图标
- 警告: 黄色 + 感叹号图标
- 信息: 蓝色 + 信息图标

### 动画
- 进入: 从右侧滑入 (300ms)
- 显示: 3-5 秒
- 离开: 淡出 (300ms)

## 加载状态

### 骨架屏
- 灰色占位符
- 脉冲动画 (1.5s)
- 保持布局稳定

### 加载动画
- 旋转圆形 (1s)
- 进度条 (线性或不确定)
- 加载文字提示

## 页面过渡

### 进入
- 淡入 (300ms)
- 从下往上滑入 (300ms)
- 缩放进入 (300ms)

### 离开
- 淡出 (300ms)
- 从上往下滑出 (300ms)
- 缩放离开 (300ms)

## 响应式交互

### 触摸设备
- 最小触摸目标: 44x44px
- 长按菜单
- 滑动手势
- 双击缩放

### 鼠标设备
- 悬停效果
- 右键菜单
- 拖拽操作
- 滚轮缩放

## 键盘导航

### Tab 顺序
- 逻辑顺序: 从左到右, 从上到下
- 跳过不可交互元素
- 模态框内循环

### 快捷键
- Enter: 确认/提交
- Escape: 关闭/取消
- Space: 激活按钮/复选框
- Arrow Keys: 导航列表/菜单
